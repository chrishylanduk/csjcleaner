How it works
  1. The SID query parameter is extracted and Base64-decoded, giving a standard query string.
  2. The decoded query string is parsed to extract the joblist_view_vac value (the job ID).
  3. A clean official URL is constructed: …/csr/jobs.cgi?jcode={jobId}.

A personal project by Chris Hyland. Not affiliated with Civil Service Jobs or any other organisation.

URL cleaning happens within your browser: nothing you enter is transmitted.
